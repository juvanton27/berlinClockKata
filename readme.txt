Le travail a été réalisé en pair programming sur le même pc avec changement de rôle toutes les 20 minutes

Le résultat en heure de Berlin sera afficher de la sorte : 0_0_0_0_0

Ces nombres désignent respectivement les secondes, les blocs de 5heures, les heures simples, les blocs de 5 minutes et les simples minutes

Pour les secondes, 1 signifie que l'ampoule est allumée et 0 éteinte

Les nombres suivants désignent le nombre d'ampoules allumées par ligne
